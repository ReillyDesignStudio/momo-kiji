# momo-kiji Roadmap

Public roadmap for momo-kiji development. Updated quarterly.

## Vision

**Make ANE (Apple Neural Engine) a first-class target for ML developers.**

Currently, ANE is invisible to most developers. CoreML locks you into Apple's ecosystem. momo-kiji changes that: bring your own model, target ANE directly, get 10x efficiency.

## Current Status

**Phase 1: Foundation (Now)**
- ✅ Core compiler infrastructure
- ✅ ANE backend generation
- ✅ Python API
- ⏳ Public launch (March 2026)
